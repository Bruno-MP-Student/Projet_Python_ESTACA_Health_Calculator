Welcome to the **Health Calculator** documentation!
===================================================

This project is designed to calculate health-related metrics such as **Body Mass Index (BMI)**.
You can use this tool to calculate BMI based on weight and height, and to classify the result into categories like "Underweight", "Normal weight", "Overweight", and "Obesity".

Features
--------
- **BMI Calculator**: Compute the BMI from weight and height.
- **BMI Category**: Classify the calculated BMI into one of the following categories: "Underweight", "Normal weight", "Overweight", and "Obesity".

Modules
-------
The **health_calculator** package contains the following modules:

- **bmi**: Functions for calculating BMI and classifying the BMI category.
